# Glassmorphic Design HTML/CSS Theme

![Glassmorphic Preview](./assets/preview.png)

A **modern, professional, and fully responsive Glassmorphic Design** template built with HTML, CSS, and a touch of JavaScript. Perfect for websites, dashboards, and UI experiments.

---

## Features

- ✅ **Glassmorphism effect** using `backdrop-filter` and semi-transparent layers   
- ✅ Minimal and **clean design** for professional projects  
- ✅ Easy to customize colors, blur, and content  
- ✅ Lightweight with no heavy dependencies
